Chewing Editor
===============================================================================

Installation
----------------------------------------------------------------------

My env is Arch Linux.


Requirements
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

QT = 5

- `qt5-tools`
- `qt5-base`

::

    pacman -Ss qt5-base qt5-tools

``qt5-tools`` will provide ``/usr/lib/qt/bin/lrelease``.
When building ``chewing-editor``, we will need it.

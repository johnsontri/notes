Cipher
----------------------------------------------------------------------

Def
    An algorithm for performing encryption

Cipher vs Code
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

Code
    - using *codebook*

    - the ciphertext contain all the information of original plaintext

Cipher
    - usually depends on a *key* (or says *cryptovariable*)

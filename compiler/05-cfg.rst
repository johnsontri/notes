Ambiguous Grammar
----------------------------------------------------------------------

Def
    對於相同 `sentence` 卻有不同的 parse tree

Example
    沒定義 operator precedence 時， :math:`1 + 2 \times 3` 怎麼 parse?

    沒 associativity 也爛掉， :math:`1 + 2 + 3`

Semantic Analysis
===============================================================================

有些東西 context-free grammer 不能處理，如: 「變數使用前要宣告」。
解法有兩種

- 用 context-sensitive grammer

- 用 context-free grammer + 有的沒的 attribute 來幫助判斷

實際上我們選 2, 實作方法有兩種
- SDD: Syntax-Directed Definition

- SDT: Syntax-Directed Translation scheme
